核心功能

What it does

逐句同步歌詞

Synced Lyrics

每秒對齊 Spotify 播放進度，歌詞唱到哪、就亮到哪，最多兩行清楚顯示。

永遠置頂懸浮

Always On Top

固定在所有視窗最上層，可自由拖曳到任何角落，不用切回 Spotify App。

工作列控制器

Taskbar Controller

常駐 Windows 工作列與系統匣，隨時可執行、顯示、隱藏或關閉 widget。

運作流程

How it works
01

授權 Spotify

CONNECT

OAuth PKCE 一次授權，安全取得播放權限，token 自動更新。

02

偵測目前歌曲

DETECT

每秒輪詢 currently-playing，掌握歌名、歌手與即時播放進度。

03

顯示同步歌詞

DISPLAY

比對時間軸，逐句點亮目前歌詞；中文 fallback 自動轉為繁體。

幾乎都找得到歌詞

Lyrics that show up
LRCLIB主要來源

高品質同步 LRC 歌詞主來源，時間軸對齊精準。

NetEase備援

主來源找不到時自動補上，中文歌詞自動轉為繁體，讓冷門歌也有字。
